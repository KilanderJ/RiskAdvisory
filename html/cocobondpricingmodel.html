CoCo Bond Pricing Model

Description

The CoCo Bond Pricing Model is a flexible tool for pricing a Contingent Convertible Bond. The model is based on a Monte Carlo Simulation of the CET1 ratio and the price of the implicit option embedded in a CoCo bond is valued as the average value over all simulations. Given some known parameters the model can imply out the value of single unkonwn parameter such as price, CET1-volatility, trigger-level etc.

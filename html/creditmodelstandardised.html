Standardised Credit Portfolio Model

Description

The Standardised Credit Portfolio Model allows you get a ready-made bond report for an existing client. If the client is a Custody client we need to get a signed approval (sekretesavstående) that we can pull data from the clients custody account. You can choose to run three types of Standardised reports: PB-report, Savings bank report and Portfolio Report. If you are dealing with a Private Banking client we suggest to use the PB-report. A database of the positions of existning clients can be found by clicking on the button below. An example of a standardised PB-report can be also be downloaded.
Position database Example report

Note: In order to have a succesful model run all instrument details must exist! If there are some details missing you will only receive an excel indata sheet back with the client's portfolio. Please fill in the missing details and then run the Customised Credit Portfolio Model instead with the PB-report selected on the Setting page.

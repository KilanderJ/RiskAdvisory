Customised Credit Portfolio Model

Description

The Customised Credit Portfolio Model allows you to run an in-depth analysis on a bond portfolio based on your choice of reports. The reports to run are selected on the Settings page of the indata template. For Private Banking clients we suggest to run only the 'PB-report' and for Financial Institutions (e.g. Savings Banks) we suggest to run the 'Periodic Savings Bank-report'. The analysis provides things such as yield to maturity, allocations over ratings and sectors, duration, stress test of rates and credit spreads, scenario analysis, VaR-calculations and much more.

  • The 'Fill indata' tab below allows you to get instrument details for a bond portfolio. You must provide ISIN-codes to make this work.
  • The 'Run Model' tab allows you to run the model and retrieve reports based on an indata sheet where all instrument details have been provided.
